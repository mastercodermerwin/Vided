
# Vided 1.0.0

Vided is a small module to do video related things like ,slow , fast and reverse...


## Usage/Examples

```python
vid = Vided()
```

Reverse
```python
vid.reverse("test_video.mp4","out_1.mp4")
```
Slow
```python
vid.slow("test_video.mp4","out_2.mp4",5)
```
Fast
```python
vid.fast("test_video.mp4","out_3.mp4",5)
```
